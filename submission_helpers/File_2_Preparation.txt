--- Experimental Setup & Approach ---
The project followed an experimental optimization approach for embedded anomaly detection using Vision Transformers. A ViT-B/16 baseline model was first implemented and benchmarked using metrics such as AUC-ROC, latency, and model size. Multiple optimization methods including pruning, token reduction (ToMe, DynamicViT), quantization (FP16, INT8), and knowledge distillation were then applied and compared. The objective was to analyze the trade-off between computational efficiency and anomaly detection performance for embedded deployment under hardware constraints.

--- Algorithm / Model Description ---
The project used a Vision Transformer (ViT-B/16) model for anomaly detection. The input image was divided into patches and processed using transformer encoder layers with self-attention mechanisms to learn image representations. The baseline model was further optimized using pruning, token reduction, quantization, and knowledge distillation for efficient embedded deployment. The ViT-B/16 model contains approximately 86 million parameters and 12 transformer encoder layers. Performance was evaluated based on anomaly detection accuracy, latency, and model size after optimization.

--- Results & Observations ---
The experiments showed that optimization methods affect accuracy and efficiency differently. Token reduction using ToMe achieved the best AUC-ROC among tested methods, while knowledge distillation produced the smallest and fastest model for deployment. FP16 quantization preserved baseline accuracy while reducing memory usage and latency. Aggressive optimization sometimes reduced anomaly detection performance, indicating the importance of balancing computational efficiency with predictive capability for embedded AI systems.
